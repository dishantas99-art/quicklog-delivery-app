# QuickLog Delivery App - Setup & Usage Guide

## Quick Start

### Prerequisites
- Node.js 18+ and npm/pnpm
- Expo CLI (`npm install -g expo-cli`)
- iOS Simulator (Mac) or Android Emulator
- Mobile device for testing (optional)

### Installation

1. **Clone the repository**
```bash
gh repo clone dishantas99-art/quicklog-delivery-app
cd quicklog-delivery-app
```

2. **Install dependencies**
```bash
pnpm install
# or
npm install
```

3. **Start the development server**
```bash
npm start
```

4. **Run on simulator/device**
- Press `i` for iOS Simulator
- Press `a` for Android Emulator
- Scan QR code with Expo Go app on mobile device

## Key Features Overview

### For Administrators

#### Dashboard
- View revenue metrics (today, month, year)
- Monitor total staff count
- Track total receipts generated
- See top items by revenue
- Quick access to all management features

#### Staff Management
- **Add Staff**: Create new staff members with unique phone numbers and PINs
- **Edit Staff**: Update staff member details (name, phone, PIN)
- **Activate/Deactivate**: Toggle staff member access
- **Delete Staff**: Remove staff members (receipts remain)
- **View Stats**: See receipts generated by each staff member

#### Export Data
- Filter receipts by date range
- Export in CSV or PDF format
- Includes staff attribution for each receipt
- Shows item count and photo count per receipt

#### View All Receipts
- Browse all receipts in the system
- Search by customer name or location
- Filter by status
- See staff who created each receipt
- View receipt totals and details

### For Staff Members

#### Personal Dashboard
- View personal receipt statistics
- See today's and this month's performance
- Monitor unsynced receipts
- Quick access to create new receipts
- Offline mode indicator

#### Create Receipt
1. **Enter Delivery Details**
   - Customer name (required)
   - Delivery location (required)
   - Date (defaults to today)

2. **Add Items**
   - Item name, quantity, and price
   - Automatic line total calculation
   - Grand total updates in real-time
   - Add/remove items as needed

3. **Add Photos** (5-7 images required)
   - Capture from camera
   - Select from gallery
   - Progress indicator shows image count
   - Remove individual images if needed

4. **Add Notes** (optional)
   - Additional delivery information
   - Free text field

5. **Submit**
   - Online: "Create Receipt" button
   - Offline: "Save Offline" button
   - Automatic sync when online

#### View Receipts
- See only receipts created by you
- Click to view full details
- See customer info, items, total, photos
- Delete receipts if needed

## Common Tasks

### Add a New Staff Member
1. Login as Admin
2. Go to Dashboard → Manage Staff
3. Tap + button
4. Enter name, phone, PIN
5. Tap "Add Staff Member"

### Create a Delivery Receipt
1. Login as Staff
2. Tap + button (FAB)
3. Fill in customer details
4. Add items with quantities and prices
5. Add 5-7 photos
6. Add optional notes
7. Tap "Create Receipt"

### Export Receipt Data
1. Login as Admin
2. Go to Dashboard → Export Data
3. Select date range
4. Choose format (CSV or PDF)
5. Tap "Generate Export"

### Deactivate a Staff Member
1. Login as Admin
2. Go to Staff Management
3. Find staff member
4. Tap "Deactivate" button
5. Staff cannot log in until reactivated

## Offline Functionality

The app works completely offline:
- All data stored locally on device
- Receipts saved even without internet
- Automatic sync when online
- Offline mode indicator shows pending items
- Manual refresh available

### Sync Status
- **Synced**: ✓ (green indicator)
- **Unsynced**: ⏳ (yellow indicator)
- **Pending**: Shows count in offline banner

## Troubleshooting

### Login Issues
- Verify phone number format (10 digits)
- Ensure PIN is exactly 4 digits
- Check if staff member is active
- Clear app cache: `expo start --clear`

### Receipt Not Saving
- Check device storage space
- Verify you have 5-7 images
- Check internet connection for sync
- Try manual refresh

### Photos Not Loading
- Check camera/gallery permissions
- Ensure sufficient device storage
- Try removing and re-adding photos
- Restart the app

### Sync Not Working
- Check internet connection
- Verify backend server is running
- Check sync queue status in offline banner
- Try manual refresh
