# QuickLog Delivery App - Features Documentation

## Overview
QuickLog is a delivery receipt management application designed for both administrators and staff members to manage delivery receipts, track revenue, and maintain staff records.

---

## Authentication

### Unified Login Screen
- **Single Login Interface**: Both Admin and Staff use the same login screen
- **Login Method**: Mobile number + 4-digit PIN
- **No Role Selection**: User role is determined by their credentials stored in the system
- **Demo Credentials**:
  - Admin: `0100000000` / PIN: `1234`
  - Staff 1: `0111111111` / PIN: `1111`
  - Staff 2: `0122222222` / PIN: `2222`

## Admin Features

### 1. Admin Dashboard
Access the main dashboard to view comprehensive business metrics.

**Revenue Tracking**:
- **Total Revenue**: Cumulative revenue from all receipts
- **Revenue by Day**: Today's revenue breakdown
- **Revenue by Month**: Current month's revenue
- **Revenue by Year**: Current year's revenue

**Key Metrics**:
- **Total Receipts**: Total number of delivery receipts generated
- **Total Staff**: Count of all active staff members
- **Today's Receipts**: Number of receipts created today
- **This Month's Receipts**: Number of receipts created this month

**Quick Actions**:
- View All Receipts
- Manage Staff
- Export Data

---

### 2. Staff Management
Complete staff member management system with full CRUD operations.

**Add Staff Member**:
- Full Name (required)
- Phone Number (required, must be unique)
- PIN (4 digits, required)
- Status: Automatically set to Active

**Edit Staff Member**:
- Update name, phone number, or PIN
- Phone number uniqueness is validated (excluding current staff member)

**Deactivate/Activate Staff**:
- Toggle staff member status between active and inactive
- Inactive staff cannot log in

**Delete Staff**:
- Remove staff member from the system
- Receipts created by deleted staff remain in the system

**Staff Information Display**:
- Staff name and phone number
- Current status (Active/Inactive)
- Number of receipts generated by each staff member

---

### 3. Export Data
Export delivery receipt data with detailed information.

**Export Features**:
- **Date Range Filter**: Select from date and to date
- **Export Formats**: CSV or PDF
- **Data Included**:
  - Receipt ID
  - Date
  - Customer Name
  - Delivery Location
  - Staff ID (who generated the receipt)
  - Total Amount
  - Status
  - Item Count
  - Image Count

**Export History**: View recently generated exports

## Staff Features

### 1. Staff Dashboard
Personal dashboard showing individual staff performance.

**Personal Statistics**:
- **Total Receipts**: Total receipts created by the logged-in staff member
- **Today's Receipts**: Receipts created today
- **This Month's Receipts**: Receipts created this month
- **Completed**: Number of completed receipts
- **Pending**: Number of pending/draft receipts

**Recent Receipts**:
- View up to 10 most recent receipts created by the staff member

---

### 2. Delivery Receipt Management

#### Create Receipt
Staff members can generate new delivery receipts with the following details:

**Delivery Details**:
- **Customer Name** (required): Name of the customer receiving the delivery
- **Location** (required): Delivery location/address
- **Date**: Date of delivery (defaults to today)

**Items**:
- Add multiple items to the receipt
- Each item includes:
  - Item Name (required)
  - Quantity (required)
  - Price per Unit (RM)
  - Automatic line total calculation
- **Grand Total**: Automatically calculated from all items
- Add/remove items dynamically

**Photos** (5-7 images required):
- **Minimum**: 5 images required per receipt
- **Maximum**: 7 images allowed per receipt
- **Image Sources**:
  - Camera: Capture photos directly
  - Gallery: Select from device photo library
- **Progress Indicator**: Shows current image count and requirement status
- Remove individual images if needed
- Images are compressed to 70% quality for storage efficiency

**Notes** (Optional):
- Additional notes or comments about the delivery
- Free text field

**Submission**:
- Online Mode: "Create Receipt" button
- Offline Mode: "Save Offline" button
- Validation ensures all required fields are completed
- Receipts are automatically synced when online

#### View Receipt Details
- Customer information and delivery location
- Itemized list with quantities and prices
- Grand total
- Photos gallery
- Notes (if any)
- Receipt status and sync status

#### Receipt Status
- **Draft**: Receipt is being created
- **Pending**: Receipt is ready but not yet finalized
- **Completed**: Receipt is finalized

## Offline Functionality

### Offline Mode Indicator
- Banner appears when device is offline
- Shows number of unsynced receipts
- Indicates changes will sync when online

### Local Storage
- All receipts are stored locally on the device
- Changes persist even without internet connection
- Automatic sync when connectivity is restored

### Sync Queue
- Pending receipts are queued for synchronization
- Automatic retry mechanism for failed syncs
- Manual refresh option available

---

## Data Storage

### Local Storage Keys
- `@quicklog_receipts`: All delivery receipts
- `@quicklog_staff`: Staff member information
- `@quicklog_sync_queue`: Pending sync items

### Receipt Schema
```typescript
{
  id: string;
  staffId: string;
  customerName: string;
  location: string;
  date: string;
  status: 'draft' | 'pending' | 'completed';
  items: Array<{
    name: string;
    quantity: string;
    price: string;
  }>;
  totalAmount: number;
  notes: string;
  images: string[];
  synced: boolean;
  createdAt: string;
}
```

## Troubleshooting

### Receipts Not Syncing
1. Check internet connection
2. Verify backend server is running
3. Check sync queue status
4. Try manual refresh

### Login Issues
1. Verify phone number format
2. Ensure PIN is exactly 4 digits
3. Check if staff member is active
4. Clear app cache and retry

### Image Upload Issues
1. Ensure you have 5-7 images
2. Check device storage space
3. Verify camera/gallery permissions
4. Try removing and re-adding images
